
**User (Користувач)**:  Сутність, що має аккаунт і може писати відгуки : зберігається у класі User

**Admin**  : User із is_admin=True, що може видаляти будь-які відгуки  : не має окремої таблиці

**Review (Відгук)** : Нотатка про книгу з назвою та текстом, прив’язана до User : зберігається у класі Review

**Author (Автор відгуку)** : User, який створив Review : встановлюється через FK

**Moderation (Модерація)** : Видалення даних із підвищеними правами  : реалізовано у delete_review()

**Session** : Збереження стану авторизації (user_id, is_admin) : через session["user_id"]
